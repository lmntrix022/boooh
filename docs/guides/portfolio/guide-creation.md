# 📘 Guide : Créer votre Portfolio "Mon Univers"

## Vue d'ensemble

Le module Portfolio vous permet de présenter vos projets, services et réalisations directement sur votre carte de visite digitale.

**Temps estimé :** 15-20 minutes

---

## ✅ Prérequis

Avant de commencer, assurez-vous d'avoir :

- [x] Un compte créé et connecté
- [x] Au moins une carte business créée (allez sur `/cards` si ce n'est pas le cas)

---

## 📋 Étapes de création

### Étape 1 : Configurer les paramètres du portfolio ⚙️

#### 1.1 Accéder aux paramètres

1. Connectez-vous à votre compte
2. Dans le menu de navigation, cliquez sur **"Portfolio"** ou allez sur `/portfolio/settings`

#### 1.2 Activer le portfolio

1. **Toggle "Portfolio activé"** : Activez-le (bouton doit être ✅ vert/bleu)
2. **Carte associée** : Sélectionnez la carte sur laquelle le bouton "Mon Univers" apparaîtra
   - Si vous n'avez qu'une carte, elle sera sélectionnée automatiquement
   - Si vous avez plusieurs cartes, choisissez celle qui hébergera votre portfolio

#### 1.3 Personnaliser l'apparence

Remplissez les champs suivants :

**Informations de base :**
- **Titre** : Le nom de votre portfolio (par défaut : "Mon Univers")
  - Exemples : "Mes Réalisations", "Portfolio", "Mes Services"
- **Sous-titre** (optionnel) : Une description courte
  - Exemple : "Designer graphique freelance depuis 2015"

**Image de couverture :**
- Cliquez sur **"Uploader une image"**
- Sélectionnez une image de couverture (max 5MB)
- L'image sera automatiquement convertie en WebP pour optimisation

**Personnalisation visuelle :**
- **Couleur principale** : Choisissez une couleur qui représente votre marque
  - Cliquez sur le sélecteur de couleur
  - Ou entrez un code hexa (ex: `#8B5CF6`)

**Options d'affichage :**
- **Afficher les catégories** : ✅ Recommandé (permet de filtrer les projets)
- **Afficher les témoignages** : ✅ Si vous avez des avis clients
- **Suivre les vues des projets** : ✅ Pour des statistiques
- **Activer les demandes de devis** : ✅ Si vous proposez des services

**Pagination :**
- **Projets par page** : Nombre de projets affichés (recommandé : 12)

**Vue par défaut :**
- **Grille** : Vue en grille (recommandé)
- **Liste** : Vue en liste
- **Mosaïque** : Vue style Pinterest

**Système de réservation (optionnel) :**
- Si vous utilisez Calendly, Cal.com, etc.
- Sélectionnez le système
- Entrez l'URL de réservation

#### 1.4 Sauvegarder

1. Vérifiez que tous les champs obligatoires sont remplis
2. Cliquez sur **"Sauvegarder les paramètres"**
3. Vous devriez voir un message de confirmation ✅

---

### Étape 2 : Créer votre premier projet 🎨

#### 2.1 Accéder à la création de projet

1. Dans le menu, cliquez sur **"Projets"** ou allez sur `/portfolio/projects`
2. Cliquez sur le bouton **"+ Nouveau Projet"**
3. Vous arrivez sur `/portfolio/projects/new`

#### 2.2 Tab "Informations" (obligatoire)

**Informations générales :**
- **Titre*** (obligatoire) : Le nom de votre projet
  - Exemple : "Refonte identité visuelle - Startup Tech"
  - Minimum 3 caractères

- **Catégorie** : Type de projet
  - Exemples : "Graphisme", "Développement Web", "Consulting", "Photographie"
  - Conseil : Créez des catégories cohérentes pour faciliter le filtrage

- **Description courte** : Résumé en une phrase (max 200 caractères)
  - Exemple : "Création d'une identité visuelle moderne pour une startup SaaS B2B"

**Tags :**
- Ajoutez des mots-clés pour décrire le projet
- Tapez un tag et appuyez sur Entrée (ou cliquez +)
- Exemples : "logo", "branding", "tech", "startup"
- Les tags apparaîtront comme filtres dans votre portfolio

**Publication :**
- **Publier le projet** : ✅ Activez pour rendre le projet visible
  - Si désactivé, le projet reste en brouillon (visible uniquement pour vous)

#### 2.3 Tab "Contenu" (recommandé)

Structure **Challenge - Solution - Résultat** pour raconter l'histoire du projet :

**Le Défi :**
- Quel était le problème du client ?
- Quel était le contexte ?
- Exemple : "Le client avait besoin d'une identité visuelle pour se démarquer dans le secteur SaaS très concurrentiel."

**La Solution :**
- Comment avez-vous résolu le problème ?
- Quelle approche avez-vous utilisée ?
- Exemple : "Création d'une charte graphique minimaliste avec des couleurs tech (bleu/violet) et un logo moderne représentant la connexion."

**Le Résultat :**
- Quels résultats avez-vous obtenus ?
- Chiffres, retours clients, impact
- Exemple : "Lancement réussi avec +40% de taux de reconnaissance de marque lors des tests utilisateurs."

#### 2.4 Tab "Médias" (important)

**Image principale (featured image) :**
1. Cliquez sur **"Uploader une image"** dans la section "Image principale"
2. Sélectionnez votre meilleure image (max 5MB)
3. L'image sera convertie automatiquement en WebP
4. Cette image apparaîtra en grand sur la fiche projet

**Galerie d'images :**
1. Cliquez sur **"+"** pour ajouter des images
2. Ajoutez plusieurs images montrant différents aspects du projet
3. Les visiteurs pourront naviguer dans la galerie
4. Conseil : 3-6 images maximum pour ne pas surcharger

**URL Vidéo (optionnel) :**
- Ajoutez un lien YouTube ou Vimeo
- Exemple : `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- La vidéo sera intégrée dans le projet

**URL PDF (optionnel) :**
- Lien vers un dossier de présentation, case study, etc.
- Exemple : `https://votre-site.com/casestudy.pdf`

#### 2.5 Tab "CTA & Témoignage" (optionnel)

**Call-to-Action :**
- **Type de CTA** : Choisissez l'action proposée aux visiteurs
  - **"Demander un devis"** : Pour des projets similaires
  - **"Réserver un RDV"** : Pour une consultation
  - **"Me contacter"** : Contact général
  - **"Lien personnalisé"** : URL custom

- **Texte du bouton** : Personnalisez le texte
  - Exemple : "Demander un projet similaire"

- **URL personnalisée** (si "Lien personnalisé" sélectionné)
  - Exemple : `https://votre-site.com/devis`

**Témoignage client (optionnel mais recommandé) :**
- **Nom du client** : Nom et titre
  - Exemple : "Marie Dupont, CEO de TechCorp"

- **Témoignage** : Citation du client
  - Exemple : "Une collaboration exceptionnelle ! L'identité créée représente parfaitement nos valeurs."

- **Note** : Évaluation de 1 à 5 étoiles
  - Sélectionnez dans le dropdown

#### 2.6 Sauvegarder le projet

1. Vérifiez que le titre est rempli (obligatoire)
2. Vérifiez que "Publier le projet" est ✅ activé
3. Cliquez sur **"Sauvegarder"** en haut ou en bas de la page
4. Vous serez redirigé vers `/portfolio/projects`
5. Votre projet apparaît dans la liste ✅

---

### Étape 3 : Créer d'autres projets (recommandé)

**Pour un portfolio complet, créez au moins 3-6 projets**

1. Répétez l'Étape 2 pour chaque projet
2. Variez les catégories et tags
3. Ajoutez des médias de qualité
4. Assurez-vous que chaque projet est **publié**

**Conseils :**
- Commencez par vos meilleurs projets
- Utilisez des images haute qualité
- Rédigez des descriptions claires et concises
- Ajoutez des témoignages quand c'est possible

---

### Étape 4 : Vérifier l'affichage public 👀

#### 4.1 Accéder à votre carte publique

1. Allez sur `/cards`
2. Cliquez sur **"Voir la carte"** de la carte où vous avez activé le portfolio
3. Ou accédez directement à `/card/{votre-card-id}`

#### 4.2 Vérifier le bouton "Mon Univers"

**Le bouton devrait apparaître** avec :
- L'icône 💼 (Briefcase)
- Le titre que vous avez configuré (ex: "Mon Univers")
- Le nombre de projets publiés (ex: "3 projets")

**Si le bouton n'apparaît pas :**
- Consultez le guide de dépannage : [DEBUG_BOUTON_MON_UNIVERS.md](DEBUG_BOUTON_MON_UNIVERS.md)
- Exécutez le script de diagnostic : [diagnose_mon_univers.sql](diagnose_mon_univers.sql)

#### 4.3 Tester la vue portfolio

1. Cliquez sur le bouton "Mon Univers"
2. Vous arrivez sur `/card/{id}/portfolio`
3. Vérifiez que :
   - ✅ Votre header personnalisé apparaît (titre, sous-titre, cover)
   - ✅ Les filtres par catégorie fonctionnent
   - ✅ Vos projets s'affichent
   - ✅ Vous pouvez cliquer sur un projet pour voir les détails
   - ✅ La galerie d'images fonctionne
   - ✅ Les CTAs et témoignages apparaissent

## 🚀 Fonctionnalités avancées

### Gestion des devis (optionnel)

1. Activez "Activer les demandes de devis" dans les settings
2. Les visiteurs peuvent demander un devis depuis :
   - Le header du portfolio
   - La fiche détail d'un projet (si CTA = "Demander un devis")
3. Consultez vos demandes sur `/portfolio/quotes`
4. Gérez le statut : Nouveau → En cours → Devis envoyé → Accepté/Refusé

### Statistiques (optionnel)

1. Activez "Suivre les vues des projets" dans les settings
2. Les vues de chaque projet seront comptabilisées
3. Consultez les stats dans la liste des projets

### Réservation en ligne (optionnel)

1. Configurez votre URL Calendly/Cal.com dans les settings
2. Un bouton "Réserver un RDV" apparaîtra dans le header du portfolio
3. Les visiteurs peuvent réserver directement
