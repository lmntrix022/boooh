# Booh

Booh is a modern web application for creating, managing, and sharing digital business cards. It features a powerful admin interface for enterprise-level management and analytics.

## Features

- **Digital Business Card Management**: Create, edit, and organize business cards for users and teams.
- **Real-Time Statistics**: View live metrics on users, cards, templates, and card views.
- **Enterprise Admin Dashboard**: Professional dashboard with status cards, trend analysis, and beautiful charts.
- **Template Editor**: Build and preview card templates with HTML, CSS, and JavaScript support.
- **User & Content Management**: Manage users, roles, templates, and platform content from a unified interface.
- **System Monitoring**: Monitor system health, performance, and resource usage in real time.
- **Role-Based Access**: Secure admin access with role management and Supabase policies.

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (database & authentication)

## Getting Started

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   cd booh-card-magic-main
   ```
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Start the development server**
   ```sh
   npm run dev
   ```
