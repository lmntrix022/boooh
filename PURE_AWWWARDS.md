# 🏆 PURE AWWWARDS - Minimalisme Extrême

## ✨ Philosophie

**Moins c'est plus. Beaucoup moins.**

- Palette : **Noir + Blanc** uniquement
- Purple : Juste pour le logo 3D (accent unique)
- Typography : **Massive et brutale**
- Espace : **Énormément de vide**
- Animations : **Subtiles mais puissantes**

---

## 🎨 Design Épuré

### Palette Monochrome
```
Blanc : #FFFFFF (sections claires)
Noir  : #000000 (sections sombres)
Gris  : #666666, #999999 (textes secondaires)
Purple: #8b5cf6 (logo 3D uniquement)
```

### Typography Brutale
```
Hero    : 12rem (192px) - ÉNORME
Titles  : 8rem (128px) - Massif
Details : 10px mono - Technique
```

### Rythme
```
Section : min-h-screen (chaque bloc = écran complet)
Spacing : Vide extrême entre éléments
Grid    : 1px borders, pas de remplissage
```

## 🎭 La Carte 3D

### Matériau
- Couleur : **Blanc** (pas bleu-violet)
- Emissive : Purple subtil (5%)
- Metalness : 90%
- Roughness : 10%

### Voyage
- Hero : Centre
- Pains : Se déplace (droite, gauche, haut, bas)
- Solutions : Retour au centre
- CTA : Zoom final

### Visibilité
- **Opacity liée au scroll** : 
  - Hero : 100%
  - Milieu : 30% (laisse place au contenu)
  - Footer : 100%

---

## ✨ Animations

### Principes
- **Lentes et fluides** (1-2 secondes)
- **Easing cubic** : [0.16, 1, 0.3, 1]
- **Pas de bounce** (trop playful)
- **Fade + Translate** uniquement

### GSAP
- Scrub: 1.5 (smooth)
- Carte voyage sans à-coups
- Damping naturel

### Framer Motion
- whileHover: scale 1.05-1.08 max
- whileInView: fade in simple
- Transitions: 1-1.5s
- No spring physics (trop bouncy)
